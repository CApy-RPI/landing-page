Get Started

CAPY has everything you need to run your club. It's features include...

Student Verification

Club leaders can quickly know if new members are students by setting up a verification process for incoming members.

Automated Attendance

Club attendance is automatically tracked and organized for club leaders.

Meeting Reminders

Reminders for club meetings can be setup so that they are automatically announced on specified days.

Event Management

Leaders can create events with details like date, time, location, and description. Members can RSVP directly through the program, helping leaders track attendance and plan accordingly.

Alumni Network

Alumni members can easily connect with current members. Some perks include posting job offerings, having dedicated channels, and helping with club recruiting.

Discord Integration

If you already have a club Discord server, you can automatically integrate the CAPY Bot to help manage your club server. If not, you can also setup a club server using CAPY.
